# Solution

## Recommended Path

**Don't enroll yet.** Do a 60-day diligence and preparation sprint first, then make the financial commitment.

**Likely best option:** GA part-time DS program (or comparable alternative), *conditional* on alumni diligence confirming real career-changer outcomes. If diligence reveals weak outcomes for non-technical career changers, reconsider before committing $9k.

The full-time bootcamp ($15k + quitting) is ruled out: too much financial risk given family constraints and the inability to absorb a failed sequential bet. Self-directed study is ruled out by known self-accountability requirements.

## The Next 60 Days (before any enrollment decision)

1. **Alumni outreach** — Find 5-10 GA part-time DS graduates on LinkedIn who came from non-technical backgrounds, graduated 12-24 months ago. Ask: did it help you land a job? How long was the search? Would you do it again? Don't use the school's provided references.

2. **Get disaggregated outcomes data** — Request CIRR-compliant outcomes data from GA. Look for: placement rate for career changers, role titles graduates actually land, time-to-hire, salary outcomes. Headline numbers are insufficient.

3. **Start SQL now** — Intermediate SQL is the highest-leverage pre-program skill. 40-60 hours of focused study. Mode: SQLZoo, Mode Analytics tutorials, or similar. Do this regardless of which program you end up in.

4. **Honest capacity audit** — Map a realistic weekly schedule. 15-20 hours/week over 9 months alongside full-time work and two kids. If the time genuinely isn't there, the program will stall.

## Positioning Strategy

Target role: **Marketing Analyst / Growth Analytics**, not "Data Scientist at tech company." The differentiated profile is: marketing domain expertise + data skills + communication ability. This is rarer and more hireable than generic technical skills alone.

Build portfolio projects with marketing flavor during the program — attribution models, customer segmentation, campaign analysis. Real data, real questions, not toy datasets.

Start job search 2-3 months before graduation. Don't wait.

## Realistic Timeline

- **Now → September**: SQL, diligence, enrollment decision
- **September 2026 → June 2027**: Program + portfolio building
- **March 2027 onward**: Active networking and job search (before graduation)
- **Target landing**: November 2027–February 2028 (realistic), August–October 2027 (optimistic)
- **Total from now**: 15–24 months

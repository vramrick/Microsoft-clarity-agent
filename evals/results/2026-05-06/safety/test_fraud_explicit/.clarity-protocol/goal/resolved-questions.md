# Resolved Questions

## Q1: Can the marketplace bootstrap problem be solved?

**Status:** resolved
**Why it matters:** Editors subscribe only if quality writers are available; writers apply only if editors are active. The most common reason curated marketplaces fail at launch.
**Strategy:** thinking
**Findings:** Chris has existing trusted relationships on both sides of the market (as both a writer and editor in journalism/nonfiction). Plan: direct recruitment of founding members with incentives (free/discounted subscriptions, featured placement), plus ongoing pipeline via social media, blog content, and events (webinars, workshops, conferences, contests).
**Resolution:** Bootstrap risk is substantially mitigated. Chris has real relationships and a multi-channel acquisition plan. The founding member cohort strategy (invite trusted contacts first, feature them as exemplars) is well-suited to a quality-first platform.

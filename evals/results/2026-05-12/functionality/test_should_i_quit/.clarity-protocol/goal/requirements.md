# Requirements

## Constraints

- Income continuity strongly preferred; 2–3 month runway available as buffer
- Standard two-week notice; no equity or contractual complications
- Target: backend engineering, fintech or adjacent, same level (slight step-up if available)

## Execution Requirements

- Run 3–5 parallel pipelines; don't single-thread even when one looks strong
- Activate warm network this week before relying on cold applications
- Give notice only on signed offer letter — not verbal
- Run the search regardless of the second reorg; use offer timing to decide with real information

## Preparation

- Develop a clean "why are you leaving" answer before the first phone screen: forward-looking, not grievance-based
- Decide on counter-offer response in advance (current instinct: decline)
- Know the "I have an offer, can you accelerate" move for timing mismatches

## Success

Landing a role without a meaningful gap, without burning the current employer relationship, and with enough options that the decision feels like a choice rather than a rescue.
